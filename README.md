# Smart Weather Monitoring System using ESP32

A compact IoT-based Smart Weather Monitoring System developed using an ESP32 microcontroller, DHT22 temperature & humidity sensor, and SSD1306 OLED display. The system continuously monitors environmental conditions and displays real-time temperature and humidity data on the OLED screen.

## Features

* Real-time temperature monitoring
* Humidity measurement using DHT22
* OLED display output
* Sensor error handling
* Serial Monitor data logging
* Low-cost and energy-efficient design

## Hardware Components

* ESP32 Development Board
* DHT22 Temperature & Humidity Sensor
* SSD1306 OLED Display
* Breadboard and jumper wires

## Software & Libraries

* Arduino IDE
* Adafruit SSD1306 Library
* Adafruit GFX Library
* DHT Sensor Library

## Applications

* Smart home automation
* Weather monitoring stations
* Environmental monitoring systems
* IoT and embedded systems projects
* Industrial temperature monitoring

## Future Enhancements

* Wi-Fi cloud monitoring
* Mobile app integration
* Automatic cooling fan control
* Data storage and analytics
* Email/SMS alert notifications

This project is ideal for beginners and IoT enthusiasts who want to learn sensor interfacing, OLED display control, and real-time monitoring using ESP32.
